# School Management System - Frontend

## Mobile-First Features

This application is optimized for mobile devices, with a focus on the Kenyan market:

### Key Mobile Features:
- **Responsive Design**: Works seamlessly on phones, tablets, and desktops
- **Touch-Optimized**: Large buttons and touch-friendly UI elements
- **Fast Loading**: Optimized for slower network connections
- **Offline Support**: PWA capabilities for offline attendance marking
- **Simple Navigation**: Easy-to-use interface for non-tech teachers

### Competitive Advantages:
1. **Mobile-First Interface**: Designed for Kenya's phone-reliant ecosystem
2. **Affordable Pricing**: Subscription-based pricing model
3. **Local Support**: Built-in helpdesk system
4. **Simple UI**: Minimal learning curve for non-technical staff

### Extra Features:
- Transport Tracking (GPS or manual)
- Hostel/Dormitory Management
- Library Management
- Inventory Management
- Timetable Generation
- ID Card QR Verification
- SMS Integration
- MPesa Payment Integration

## Running the Application

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

## Build for Production

```bash
npm run build
npm start
```
