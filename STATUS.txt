Core Features:
1. Fee Tracking System (with overpayment & carryover)
2. MPesa Integration (Daraja API with STK Push)
3. Student Management (admission, enrollment, IDs)
4. Parent Portal (fees, payments, results)
5. Teacher Portal (attendance, marks, classes)
6. SMS Communication (reminders, confirmations)
7. Exam & Report Card System (PDF generation)
8. Admin Dashboard (analytics, summaries)
9. Role-Based Access Control (6 roles)
10. Offline Support (PWA with service workers)

Extra Features:

3. Library Management (books, borrowing)
4. Inventory Management (stock tracking)
5. Timetable Generation (weekly schedules)
6. QR Code ID Verification (student IDs)
